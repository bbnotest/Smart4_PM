# 🖥️ LMS 사용법 (HTML 학습 포털)

이 폴더의 **`.md` 가이드 문서를 브라우저에서 보고·관리**하는 간단한 LMS입니다.
별도 설치(서버/DB) 없이, **로컬 서버만 띄우면** 바로 동작합니다. 콘텐츠 원본은 계속 `.md` 파일이며, 그 파일을 그대로 읽어 화면에 렌더링합니다.

---

## ▶ 실행 방법 (셋 중 하나)

### 방법 1 — `start.bat` 더블클릭 (가장 쉬움, Windows)
1. 이 폴더의 **`start.bat`** 을 더블클릭
2. 검은 창이 뜨고, 잠시 후 브라우저에서 `http://localhost:8000/` 가 열립니다
3. 빈 화면이면 **F5(새로고침)** 한 번
4. 종료: 검은 창에서 **Ctrl + C** 후 창 닫기

> `start.bat` 은 Python → Node.js(npx serve) 순으로 자동으로 사용 가능한 도구를 찾습니다.

### 방법 2 — 직접 명령어 (Python이 있다면)
PowerShell/터미널을 이 폴더에서 열고:
```powershell
python -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000/` 접속.

### 방법 3 — VS Code "Live Server"
VS Code로 이 폴더를 열고, `index.html` 우클릭 → **Open with Live Server**.

> ⚠️ **`index.html` 을 그냥 더블클릭(file://)하면 안 됩니다.** 브라우저 보안 정책상 `.md` 파일을 읽지 못해 빈 화면이 됩니다. 반드시 위 "로컬 서버" 방식으로 여세요.

## ✨ 기능

| 기능 | 설명 |
|---|---|
| 📑 **문서 렌더링** | `.md` 를 자동 변환 — 표·코드·**Mermaid 다이어그램**·**SVG 목업**까지 표시 |
| 🗂️ **사이드바 내비** | 시작하기 / 툴별(Trello·Jira·Asana·Redmine) / 마무리 구조로 자동 정리 |
| ✅ **진도 체크** | 문서별 "완료 표시" + 상단·대시보드 **전체 진도율** |
| ☑️ **연습 체크리스트** | 직접 해보기 문서의 `- [ ]` 항목을 **클릭해 체크**, 자동 저장 |
| 🔎 **검색** | 상단 입력창: 제목 필터 / **Enter = 전체 본문 검색** |
| ◀▶ **이전·다음** | 문서 하단에서 순서대로 이동 |
| 🌙 **라이트/다크** | 우측 상단 토글 |
| 🏠 **대시보드** | 툴별 카드 + 직접 만들어보기 + 진도 |

진도·체크 기록은 **이 브라우저의 localStorage** 에 저장됩니다(개인 단위). 초기화는 사이드바 하단 **"↺ 진도 초기화"**.

## 🛠️ 문제 해결

| 증상 | 원인/해결 |
|---|---|
| 빈 화면, "콘텐츠를 표시할 수 없습니다" | `file://` 로 열었음 → **start.bat / 로컬 서버**로 열기 |
| 다이어그램·표가 안 보임 | 최초 1회 **인터넷 필요**(marked/mermaid CDN). 사내망 차단 시 → 아래 "오프라인" |
| 브라우저가 안 열림 | `http://localhost:8000/` 수동 접속. 포트 충돌 시 `start.bat`의 `PORT` 수정 |
| 진도가 사라짐 | 다른 브라우저/시크릿창은 별도 저장. 같은 브라우저로 접속 |

### 오프라인(사내망)에서 쓰려면
`index.html` 상단의 CDN 스크립트(marked·mermaid·highlight.js)를 한 번 내려받아 `lms/vendor/` 에 두고, `index.html`의 `src`/`href` 를 로컬 경로로 바꾸면 인터넷 없이도 동작합니다. (필요 시 도와드릴 수 있습니다.)

## 📁 구성 파일

```
PM/
├─ index.html          ← LMS 진입점 (로컬 서버로 열기)
├─ start.bat           ← 원클릭 실행 런처
├─ LMS_사용법.md       ← (이 문서)
├─ lms/
│   ├─ curriculum.js   ← 문서 목록/구조 정의 (문서 추가 시 여기 수정)
│   ├─ styles.css      ← 디자인/테마
│   └─ app.js          ← 동작 로직
└─ (기존 .md 가이드 + assets/ 목업)
```

> **문서(가이드)를 추가**하려면: `.md` 파일을 만들고 → `lms/curriculum.js` 의 해당 섹션 `items` 에 `{ path, title, type }` 한 줄을 추가하면 사이드바에 자동 반영됩니다.
