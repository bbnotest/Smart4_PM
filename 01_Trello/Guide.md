## 🧩 Trello가 뭐예요?

화이트보드에 **포스트잇을 붙였다 떼었다** 하는 도구예요. 일이 진행되면 포스트잇을 오른쪽 칸으로 옮깁니다. 그게 전부입니다.

기억할 단어는 **딱 4개** (모두 무료):

![Trello 구조](../assets/trello_structure.svg)

| 단어 | 뜻 |
|---|---|
| **Board**(보드) | 프로젝트 1개 |
| **List**(리스트) | 진행 단계 (할 일 → 하는 중 → 완료) |
| **Card**(카드) | 작업 1개 |
| **Label**(라벨) | 색으로 분류 |

> ⭐ 무료로 **칸반 보드 전체 + Butler 자동화(월 250회)** 가 열립니다. 단, **Calendar·Timeline·Table·Dashboard 뷰는 Premium** — 그건 8단계에서 정리해요.

---

## 🎯 이 가이드를 끝내면

- [ ] 보드를 만들고 워크플로(리스트)를 짤 수 있다
- [ ] 카드로 작업을 등록하고 라벨·체크리스트·마감으로 꾸밀 수 있다
- [ ] 카드를 옮기고 필터로 보드를 운영할 수 있다
- [ ] Butler 자동화를 1개 만들 수 있다
- [ ] 무료/Premium 경계를 설명할 수 있다
