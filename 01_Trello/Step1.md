# 🟦 Trello · 1단계 — 계정과 보드 만들기

> 🎯 이번 단계 목표: **가입하고, 첫 보드를 만든다.** (약 7분)
> 📍 [← 개요](Guide.md) · 다음 [2단계 →](Step2.md)

---

## A. 계정 만들기 (3분)

1. 브라우저(크롬 권장)에서 **https://trello.com/signup** 접속
2. **`Continue with Google`** 로 가입 (이메일 가입도 OK)
3. 워크스페이스 이름을 물으면 `GameDev Academy` 입력

> 🙋 **영어가 부담되면** 크롬에서 우클릭 → "한국어로 번역"을 켜세요. 버튼 위치는 같습니다.

> 🖼️ 공식 스크린샷 자리 — Trello 가입 화면
> 출처: https://support.atlassian.com/trello/docs/creating-a-new-board/

---

## B. 보드 만들기 (2분)

보드 = **프로젝트 하나**를 담는 화이트보드입니다.

1. **`Create`**(만들기) 버튼 클릭 → **`Create board`**
2. 제목에 `Pixel Dungeon Run - Sprint 1` 입력
3. **`Create`** 클릭

![Trello 구조](../assets/trello_structure.svg)

> 위 그림처럼, 앞으로 이 **보드** 안에 리스트와 카드를 채워갑니다.

> 🖼️ 공식 스크린샷 자리 — 보드 생성 다이얼로그
> 출처: https://trello.com/guide/create-project

---

## ✅ 확인

- [ ] 화면 왼쪽 위에 워크스페이스 이름이 보인다
- [ ] 비어 있는 보드 화면이 열렸다

> ℹ️ 무료는 워크스페이스당 **보드 10개·협업자 10명** 까지. 연습엔 충분합니다.
