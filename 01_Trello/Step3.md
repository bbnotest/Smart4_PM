## 카드 = 작업 1개

1. `Backlog` 리스트 아래 **`+ Add a card`** 클릭
2. 아래 9개를 **제목만** 한 줄씩 입력 (Enter로 연속 추가):

| 카드 제목 |
|---|
| US-01 플레이어 이동(턴제) |
| US-02 공격 입력 |
| US-03 이동 처리 |
| US-04 턴제 전투(HP) |
| US-05 절차적 던전 생성 |
| US-06 적 추적 AI |
| US-07 사망/게임오버 화면 |
| US-08 핵심 효과음 |
| US-09 1층 플레이 프로토타입 빌드 |

완성하면 이런 모습이 됩니다 👇

![Trello 보드 목업](../assets/trello_board_mockup.svg)

## 🎮 현장 감각 — 게임 PM은 이렇게

> **Pixel Dungeon 맥락** — 먼저 **머릿속을 다 쏟아내(braindump)** Backlog에 카드로 쌓는 게 1순위입니다. 다듬기는 나중. 카드 제목을 **유저 스토리(US-xx)** 로 적으면 나중에 Jira로 옮길 때도 그대로 이어집니다.

**⚠️ 흔한 실수**
- 카드 하나에 **여러 작업**을 욱여넣음 → 카드 = **작업 1개**.
- 완벽하게 적으려다 시작을 못 함 → 일단 제목만, 상세는 4·5단계에서.

**🎤 면접 한 줄**
> *"기능을 **유저 스토리 카드**로 백로그에 정리해, 무엇을 만들지 한눈에 보이게 했습니다."*
