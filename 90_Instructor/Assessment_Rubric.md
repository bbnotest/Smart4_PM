# 📝 평가 루브릭 (Assessment Rubric)

> 산출물을 일관되게 채점하기 위한 기준표. **자기평가 → 동료 리뷰 → 강사 평가** 3단계에 공통 사용합니다.

---

## 1. 배점 개요 (총 100%)

| 영역 | 산출물 | 배점 |
|---|---|:--:|
| Trello | Sprint 1 Kanban 보드 | 15 |
| Jira | 백로그+Sprint 1 **및** Timeline | 15 |
| Asana | 다중 뷰 프로젝트 | 15 |
| Redmine | 프로젝트 + Gantt | 15 |
| **캡스톤** | 툴 선택 보고서 + 미니 계획 + 발표 | **30** |
| 참여/회고 | 출석·토론·회고 | 10 |
| | **합계** | **100** |

> Jira는 D2(백로그+스프린트)와 D3a(Timeline) 두 산출물을 합쳐 **하나의 15점**으로 평가합니다.

---

## 2. 공통 평가 척도

| 수준 | 점수 비율 | 의미 |
|---|:--:|---|
| **탁월** | 100% | 기준을 모두 충족 + 도전 과제/통찰까지 |
| **우수** | 85% | 기준을 충실히 충족 |
| **보통** | 70% | 핵심은 했으나 일부 누락/오류 |
| **미흡** | 50% | 절반 이하 충족, 재제출 권장 |
| 미제출 | 0% | — |

---

## 3. 툴별 산출물 루브릭 (각 15점)

### 🟦 Trello (D1)
| 기준 | 배점 | 탁월 | 보통 |
|---|:--:|---|---|
| 워크플로 구성 | 3 | 5개 리스트가 논리적 순서 | 단계 일부 누락 |
| 작업 카드 | 3 | 9개 카드 + 제목 명확 | 일부 누락 |
| 분류/세부 | 3 | 라벨(에픽)+체크리스트+마감 | 라벨만 |
| 칸반 운영 | 3 | 상태별 적절히 분산 | 전부 한 리스트 |
| 자동화 | 3 | Butler 규칙 동작 | 없음/오작동 |

### 🟦 Jira (D2+D3a)
| 기준 | 배점 | 탁월 | 보통 |
|---|:--:|---|---|
| 이슈 계층(WBS) | 3 | 에픽7+스토리9+서브태스크 | 에픽 연결 누락 |
| 추정/메타 | 3 | 포인트·담당·우선순위 완비 | 일부만 |
| 스프린트 | 3 | 15pt로 구성·**시작**됨 | 시작 안 함 |
| Timeline | 3 | 에픽 일정+마일스톤+의존성 | 막대만 |
| 운영/리포트 | 3 | 보드 이동+번다운 이해 | 정적 |

### 🟧 Asana (D3b)
| 기준 | 배점 | 탁월 | 보통 |
|---|:--:|---|---|
| 구조 | 3 | 섹션5+태스크9 | 섹션 미사용 |
| 서브태스크 | 3 | US-05 3개 | 없음 |
| 커스텀필드 | 3 | Priority 전 태스크 | 제목에만 표기 |
| 다중 뷰 | 3 | List/Board/Calendar 정상 | 1개 뷰만 |
| 마일스톤 | 3 | 2개 이상 ◆ | 일반 태스크로 |

### 🟥 Redmine (D4)
| 기준 | 배점 | 탁월 | 보통 |
|---|:--:|---|---|
| 환경 구축 | 3 | 직접 띄워 로그인 | 공용서버 의존 |
| 상위/하위 이슈 | 3 | 2상위+5하위 WBS | 평면 나열 |
| 날짜/진행률 | 3 | 시작·마감·%done 완비 | 누락多 |
| 버전/로드맵 | 3 | M1·M2+Target version | 버전만 |
| Gantt | 3 | 막대·진행률·버전선 표시 | 비어있음 |

## 4. 캡스톤 루브릭 (30점)

| 기준 | 배점 | 탁월 | 보통 |
|---|:--:|---|---|
| **상황 분석** | 5 | 제약을 정확히 짚음 | 피상적 |
| **툴 선택 근거** | 8 | 비교 우위로 3+근거, 탈락툴 이유 명확 | 단순 선호 |
| **약점·보완책** | 5 | 한계 인지+현실적 보완 | 약점 미인지 |
| **미니 계획 완성도** | 7 | 에픽3+작업6+마일스톤+시그니처 기능 | 최소요건 미달 |
| **발표/전달** | 5 | 5분 내 설득력+시연 | 시간초과/모호 |

> 캡스톤의 핵심은 **"왜 다른 툴을 안 골랐는가"의 논리**. 선택 자체가 아니라 근거를 본다.

## 5. 참여/회고 (10점)

| 기준 | 배점 |
|---|:--:|
| 출석·실습 참여 | 4 |
| 비교 토론 기여 | 3 |
| 일자별 회고 작성(4툴 비교 통찰) | 3 |

---

## 6. 채점 가이드

- 각 산출물은 학생용 `Practice.md`의 **완성 기준 체크리스트** 통과가 '보통(70%)' 기준선.
- 도전 과제 수행/툴 간 비교 통찰이 보이면 '탁월'.
- **재제출**: 미흡(50%)은 피드백 후 1회 재제출 기회 권장.

### 종합 등급(참고)
| 총점 | 등급 |
|---|---|
| 90–100 | A (현업 투입 가능 수준) |
| 80–89 | B (실무 적응 빠름) |
| 70–79 | C (기본기 확보) |
| <70 | 보강 필요 |
