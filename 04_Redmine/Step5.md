# 🟥 Redmine · 5단계 — 내장 간트 차트

> 🎯 이번 단계 목표: **날짜와 진행률을 넣어 무료 간트를 완성한다.**
> 📍 [← 4단계](Step4.md) · 다음 [6단계 →](Step6.md)

---

간트는 이슈의 **시작일 · 마감일 · 진행률(% Done)** 로 자동으로 그려집니다.

## 입력하기

1. 각 하위 이슈를 열어 **`Start date`**(시작) · **`Due date`**(마감) 입력
2. **`% Done`**(진행률) 도 입력

| 이슈 | 시작 | 마감 | %Done |
|---|:--:|:--:|:--:|
| US-01 | 7/06 | 7/12 | 60 |
| US-02 | 7/06 | 7/19 | 30 |
| US-04 | 7/13 | 7/19 | 0 |
| US-05 | 7/20 | 7/31 | 0 |
| US-06 | 7/27 | 8/02 | 0 |

3. 상단 **`Gantt`** 탭 클릭 → 막대·진행률·상위하위·버전선이 표시됩니다

완성하면 이런 모습입니다 👇

![Redmine 내장 Gantt 목업](../assets/redmine_gantt_mockup.svg)

> 🙋 **간트가 비어 보이면** 날짜를 안 넣은 것 → **시작/마감 날짜가 있어야** 막대가 그려집니다.
> 💡 Redmine 간트는 **드래그 편집이 안 됨**(보기 위주). 일정 변경은 이슈의 날짜를 수정하면 반영됩니다.

> 🖼️ 공식 스크린샷 자리 — Gantt

- [ ] Gantt 탭에 막대·진행률·버전선이 보인다
